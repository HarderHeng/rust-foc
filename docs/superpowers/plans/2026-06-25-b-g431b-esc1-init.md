# B-G431B-ESC1 Embassy Project Init Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring up an Embassy Rust firmware skeleton on ST B-G431B-ESC1 that initializes USART2 (PB3 TX, PB4 RX) for a future debug shell, integrates defmt logging, and proves system health via a 500ms heartbeat.

**Architecture:** Single Cargo binary crate, multi-module layered (BSP → Driver → Tasks). `Uart2Sink` implements both `DebugShellSink` (app-level) and `embedded_io::Write` (third-party lib compatibility). Defmt dual-channel: RTT for dev, USART2 for shell.

**Tech Stack:**
- Embassy (stm32, executor, time) 0.x
- embedded-io 0.7
- defmt 0.3 + defmt-rtt 0.4
- cortex-m / cortex-m-rt 0.7
- panic-probe 0.3
- target: `thumbv7em-none-eabihf`
- runner: probe-rs

**Spec:** `docs/superpowers/specs/2026-06-25-b-g431b-esc1-initialization-design.md`

These come from the spec and apply to every task. Read them once; they don't change.

- **MCU:** STM32G431CBU6 (128KB flash @ 0x08000000, 32KB SRAM @ 0x20000000)

- **Layered rule:** `src/tasks/*` MUST NOT `use embassy_stm32::...`. Only `use crate::drivers::debug_uart::{DebugShellSink, Uart2Sink}`.
- **Dependency versions:** All `"*"` in Cargo.toml EXCEPT `embedded-io = "0.7"` and `embedded-cli = "0.2.1"` (locked for next spec). Resolve to current latest at first `cargo build`.
- **Release profile:** `opt-level = "s"`, `lto = true`, `codegen-units = 1`, `strip = true`, `debug = 2`
- **Code style:** Rust 2024 edition; `snake_case` for items, `CamelCase` for types, `SCREAMING_SNAKE_CASE` for consts

```rust
//! Debug serial output driver for USART2.
//!
//! `Uart2Sink` wraps a `BufferedUart` and implements both:
//! - [`DebugShellSink`]: high-level `&str` writing for app tasks
//! - `embedded_io::Write`: standard interface for third-party libs (e.g. `embedded-cli`)
//!
//! All blocking semantics: this sink writes synchronously to a TX ringbuffer.
//! Long blocking is only possible if the ringbuffer is full.

use embedded_io::Error as _Error;

/// High-level sink for debug shell output.
///
/// Tasks should depend on this trait, not on the concrete USART type.
/// This keeps the application layer independent of the transport.
pub trait DebugShellSink {
    type Error: core::fmt::Debug;

    /// Write a string slice to the sink. Returns the number of bytes written.
    /// Default implementation forwards to the implementor's `embedded_io::Write`.
    fn write_str(&mut self, s: &str) -> Result<(), Self::Error>;
}

/// USART2-backed debug sink.
///
/// `'d` is the lifetime of the USART peripheral borrowed from `embassy-stm32`.
/// The type parameter `U` is the underlying buffered UART type (left as
/// `?Sized` for now; in Task 5 we tighten it).
pub struct Uart2Sink<'d, U: ?Sized> {
    inner: &'d mut U,
}

impl<'d, U: ?Sized> Uart2Sink<'d, U> {
    /// Construct a sink from a borrowed buffered UART.
    /// Will be wired to actual `BufferedUart` in Task 5.
    pub fn new(inner: &'d mut U) -> Self {
        Self { inner }
    }
}

impl<'d, U: embedded_io::Write + ?Sized> DebugShellSink for Uart2Sink<'d, U> {
    type Error = U::Error;

    fn write_str(&mut self, s: &str) -> Result<(), Self::Error> {
        // We delegate to the underlying `Write::write`. To avoid losing bytes
        // on partial writes, loop until everything is sent or an error occurs.
        let mut written = 0;
        while written < s.len() {
            match self.inner.write(&s.as_bytes()[written..]) {
                Ok(0) => break, // writer can't accept more right now
                Ok(n) => written += n,
                Err(e) => return Err(e),
            }
        }
        Ok(())
    }
}

impl<'d, U: embedded_io::Write + ?Sized> embedded_io::Write for Uart2Sink<'d, U> {
    type Error = U::Error;

    fn write(&mut self, buf: &[u8]) -> Result<usize, Self::Error> {
        self.inner.write(buf)
    }

    fn flush(&mut self) -> Result<(), Self::Error> {
        self.inner.flush()
    }
}
```

```rust
//! Board Support Package for ST B-G431B-ESC1.

use embassy_stm32::{
    bind_interrupts,
    mode::Async,
    peripherals::{PB3, PB4, USART2},
    usart::{BufferedUart, Config, Uart},
    Config as HalConfig, Peripherals,
};
use embassy_time::Duration;

use crate::drivers::debug_uart::Uart2Sink;

pub const BOARD_NAME: &str = "B-G431B-ESC1";
pub const BOARD_MCU: &str = "STM32G431CBU6";

pub const DEBUG_UART_BAUD: u32 = 115_200;
pub const DEBUG_UART_TX_PORT: char = 'B';
pub const DEBUG_UART_TX_PIN: u8 = 3;
pub const DEBUG_UART_RX_PORT: char = 'B';
pub const DEBUG_UART_RX_PIN: u8 = 4;
pub const DEBUG_UART_AF: u8 = 7;

pub const DEBUG_UART_TX_BUF_SIZE: usize = 256;
pub const DEBUG_UART_RX_BUF_SIZE: usize = 64;

// Static buffers for the ringbuffers — must be `'static` so the
// BufferedUart can outlive the BSP scope and be moved into a task.
static mut DEBUG_UART_TX_BUF: [u8; DEBUG_UART_TX_BUF_SIZE] = [0; DEBUG_UART_TX_BUF_SIZE];
static mut DEBUG_UART_RX_BUF: [u8; DEBUG_UART_RX_BUF_SIZE] = [0; DEBUG_UART_RX_BUF_SIZE];

bind_interrupts!(struct Irqs {
    USART2 => embassy_stm32::usart::InterruptHandler<USART2>;
});

pub struct BoardHandles {
    pub debug_uart: Uart2Sink<'static, BufferedUart<'static, USART2>>,
}

pub fn board_init(p: Peripherals) -> BoardHandles {
    // SAFETY: This is a single-threaded (pre-executor) init function;
    // no other code has access to these statics yet.
    let tx_buf = unsafe { &mut *(&mut DEBUG_UART_TX_BUF as *mut _) };
    let rx_buf = unsafe { &mut *(&mut DEBUG_UART_RX_BUF as *mut _) };

    let mut cfg = Config::default();
    cfg.baudrate = DEBUG_UART_BAUD;

    let uart = Uart::new(
        p.USART2,
        p.PB3,  // TX
        p.PB4,  // RX
        Irqs,
        p.DMA1_CH1,  // TX DMA channel; if embassy complains, try DMA1_CH2
        p.DMA1_CH2,  // RX DMA channel
        cfg,
    )
    .unwrap();

    let buffered = BufferedUart::new(
        uart,
        tx_buf,
        rx_buf,
    );

    BoardHandles {
        debug_uart: Uart2Sink::new(buffered),
    }
}
```

> **Note on DMA channels:** STM32G431 has `DMA1` with 6 channels. USART2 TX/RX on this MCU are documented as supporting DMA. Common pairings: `DMA1_CH1` (TX) and `DMA1_CH2` (RX), but embassy-stm32 will tell you the correct one at compile time. If `p.DMA1_CH1` errors, try `p.DMA1_CH2` / `p.DMA1_CH3` until it compiles.

> **Note on the bind_interrupts import path:** embassy-stm32's `bind_interrupts!` macro is re-exported from the crate root. If `bind_interrupts!` macro isn't found, try `embassy_stm32::bind_interrupts!` or use the full path.

```rust
use defmt::info;
use embassy_time::Timer;

use crate::drivers::debug_uart::{DebugShellSink, Uart2Sink};
use embassy_stm32::usart::BufferedUart;
use embassy_stm32::peripherals::USART2;

#[embassy_executor::task]
pub async fn heartbeat(mut sink: Uart2Sink<'static, BufferedUart<'static, USART2>>) {
    let mut tick: u32 = 0;
    loop {
        tick = tick.wrapping_add(1);
        // Note: write_str is sync (synchronous into ringbuffer).
        // For 500ms cadence and small strings, this is fine.
        let _ = sink.write_str("[hb] tick\n");
        info!("heartbeat tick={}", tick);
        Timer::after_millis(500).await;
    }
}
```

```rust
    }
```
